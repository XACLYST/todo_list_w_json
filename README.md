# TODO LIST PROGRAM USING JSON TO SAVE DATA

A small pet-project in C# to use json from the C# code.
This program takes user's input to make description of a new task

It has 4 options:
- Make new record
- Show all records
- Mark a record as "done"
- Delete a record

All the data is being saved in the file /todo_w_json/data.json

Structure of data:

- id
- description
- completion
- date
